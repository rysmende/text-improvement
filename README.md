## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/rysmende/text-improvement.git
   ```
2. Open the repo
   ```sh
   cd text-improvement
   ```
3. Install dependencies
   ```sh
   pip install -r requirements.txt
   ```


## Usage

   ```sh
   python3 suggest.py -t data/sample_text.txt
   ```
